关于 Cocosoys

我是 Cocosoys，一个偏独立开发者和创作者方向的玩家。这个站点会记录我做过的插件、工具、接口兼容、服务器玩法实验，以及一些带有 ACG 氛围的创作想法。

我更喜欢把项目做成“能被继续使用的小工坊”：不追求大而全，优先解决具体问题，保留足够清楚的结构，让以后回来看时还能继续接上。

我常做的事

  • Minecraft / Bukkit 插件：围绕旧版本生态、服务器玩法、指令和属性扩展做轻量工具。
  • 命令与自动化：把重复操作整理成可配置、可复用的流程。
  • 兼容与接口：给已有生态补一层稳定 API，让旧环境也能跑得更舒服。
  • 创作实验：把笔记、解谜、设定和系统设计混在一起，做一点像“工坊档案”的东西。

项目关键词

Java、Bukkit、Minecraft、Node.js、Hexo、GitHub、旧版本兼容、服务器工具、创作笔记

联系与入口

  • GitHub: github.com/cocosoys
  • 站点: cocosoys.github.io

如果你是从某个项目点进来的，可以先看首页的项目卡片；如果你想了解完整列表，可以去 GitHub 或阅读项目手记。
