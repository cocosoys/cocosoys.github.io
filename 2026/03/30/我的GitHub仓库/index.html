我的GitHub仓库

我的GitHub公开仓库

以下是我的GitHub公开仓库列表：

  • AgainRunCommand - 指令规则,可以自定义指令之间联发
  • ArcaneResearch - 致力于打造一款以研究笔记解密为核心，以笔记的内容探讨万物的内核
  • AutoSellForPlayer - 自动将玩家捡起的物品出售
  • CheckCommandSource - 查询注册指令的插件名称以及插件路径
  • CValue - Provide some APIs for JRMCore
  • DoubleMineral - 简单的时运方块效果,仅支持1.8.8
  • HealthCMD - 该插件提供一个控制玩家生命值百分比指令，可以通过设置百分比来控制玩家的生命值
  • SXAttributeExtraDigRange - 增添一个SX属性：DIgRange，使手持物品能够范围破坏方块
